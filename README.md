# Storybook Resume

An interactive, scroll-driven autobiography website that tells your professional journey through animation and storytelling.

## Overview

Storybook Resume is a modern, animated portfolio website that transforms your career trajectory into an engaging visual narrative. As users scroll through the page, they experience your story unfold through synchronized animations, creating an immersive and memorable way to showcase your professional journey.

## Features

- **Scroll-Driven Animations**: All animations are synchronized with user scroll position using GSAP ScrollTrigger
- **Mobile-First Design**: Fully responsive with optimized experiences for both mobile and desktop devices
- **Interactive Elements**: Dynamic clouds, trains, bikers, and cityscapes that animate as you scroll
- **Story Sections**:
  - **Hero**: Opening introduction
  - **About**: Personal background and story
  - **Geography**: Location and regional context
  - **Education**: Academic journey
  - **Professional**: Career milestones and experience
  - **Today**: Current status and future goals

## Technologies

- **React** (18+)
- **GSAP** (GreenSock Animation Platform)
  - `gsap` v3.13.0+
  - `@gsap/react` v3.13.0+
  - `ScrollTrigger` plugin
- **CSS3** with custom properties and animations
- **Google Fonts** (Audiowide)

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Basic knowledge of React

## Installation

1. Clone the repository:
```bash
git clone <your-repository-url>
cd storybook-resume
```

2. Install dependencies:
```bash
npm install
```

3. Install GSAP packages:
```bash
npm install gsap @gsap/react
```

4. Start the development server:
```bash
npm start
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser
